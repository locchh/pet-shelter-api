# Pet Shelter API

An Express REST API built with TypeScript for managing a pet shelter.

## Tech Stack

- **Node.js** with **TypeScript**
- **Express 5**

## Project Structure

```
src/         # TypeScript source files
dist/        # Compiled JavaScript output
```

## Setup

Install dependencies:

```bash
npm install
```

## Development

Compile TypeScript:

```bash
npx tsc

# Or setup package.json with "build": "npx tsc"
npm run build
```

Run the server:

```bash
node dist/index.js

# Or compile and start in one command
npm start
```

The server listens on **port 8000**.

## Initializing a New Project

```bash
npm init -y
npx tsc
```
